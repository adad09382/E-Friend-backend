# Schema Design

## User

- \_id: ObjectId // Unique identifier generated by MongoDB
- account: String
- email: String
- password: String
- tokens: String // JWT Token
- role: Number // Admin or User

## Prompt

- \_id: ObjectId // Unique identifier generated by MongoDB
- topic: String,
- content: String,
- image: String,

## Conversation

- \_id: ObjectId // Unique identifier generated by MongoDB
- user_id: String,
- history: [{
  role:String,
  content: String,
  audioLink: { type: String, required: false }, // optional audio link
  }],
